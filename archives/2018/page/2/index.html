(String),new String(),toString,String.valueOf

  • March 24, 2018
  • v

(String)Object

强制类型转换,只有当Object引用了String类型时,运行才不会出错
注意:可以使用(String)null,但还是返回null.而不是”null”

new String(param)

生成一个新的String对象,可以传入的param参数:
byte[] bytes
char[] value
String original
StringBuffer buffer
StringBuilder builder

toString

调用Object.toString()方法,返回该对象的字符串表示,通常该字符串由类名+”@”+对象的哈希码组成
但是调用toString()方法的对象不能为null,否则会抛出异常:java.lang.NullPointerException
