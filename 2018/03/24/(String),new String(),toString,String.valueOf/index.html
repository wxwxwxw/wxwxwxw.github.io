(String),new String(),toString,String.valueOf

  • 2018-03-24
  • v

(String)Object

强制类型转换,只有当Object引用了String类型时,运行才不会出错
注意:可以使用(String)null,但还是返回null.而不是”null”

new String(param)

生成一个新的String对象,可以传入的param参数:
byte[] bytes
char[] value
String original
StringBuffer buffer
StringBuilder builder

toString

调用Object.toString()方法,返回该对象的字符串表示,通常该字符串由类名+”@”+对象的哈希码组成
但是调用toString()方法的对象不能为null,否则会抛出异常:java.lang.NullPointerException

String.valueOf(Object)

查看源码

发现它本质上就是调用Object.toString()方法,但是会判断是否为空,如果为空,则返回”null”字符串
注意:是字符串”null”,而不是null对象

除了Object,String.value()还可以传入其他参数:
boolean b
char c
double d
float f
int i
long l
char[] data
