base64编码解码

  • 2018-04-01
  • v

方式一
com.sun.org.apache.xerces.internal.impl.dv.util.Base64

byte[] data;
编码:
return String
com.sun.org.apache.xerces.internal.impl.dv.util.Base64.encode(data);
解码:
return byte[]
com.sun.org.apache.xerces.internal.impl.dv.util.Base64.decode(str);

方式二
org.apache.commons.codec.binary.Base64

byte[] data;
编码:
return byte[]
Base64.encodeBase64(data);
return String
Base64.encodeBase64String(data);
解码:
return byte[]
Base64.decodeBase64(str);

方式三 (有长度限制,不建议使用 rfc规范:76个字符换一次行)
sun.misc.BASE64Encoder

byte[] data;
编码:
return String
BASE64Encoder bs = new BASE64Encoder();
bs.encode(data);
解码:
return byte[]
BASE64Decoder bs = new BASE64Decoder();
bs.decode(str);
