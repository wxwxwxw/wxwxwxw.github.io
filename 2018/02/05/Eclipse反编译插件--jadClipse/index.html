Eclipse反编译插件--jadClipse

  • 2018-02-05
  • v

JadClipse是一个插件,可以将Jad(快速Java反编译器)与Eclipse无缝集成

在默认配置下,eclpse只能查看jre下包的源码,安装了jadClipse之后,我们就可以查看所有包的源码

下载

下载jadClipse

下载地址: 地址
下载得到一个jar文件(net.sf.jadclipse_3.3.0.jar)

下载jad

下载地址: 地址
下载得到一个压缩文件,解压后得到一个exe文件(jad.exe)

配置

打开eclipse安装目录,打开目录下的dropins文件夹,新一个jad文件夹,把上面我们得到的exe文件放进来(可以随便放,但是建议放这里),再新建两个文件夹features和plugins,把上面我们下载的jar文件放到plugins文件夹里面

然后打开eclipse,点击Window-Prefences-Java就能看到JadClipse了
在Path to decompiler一栏中填入jad.exe的路径,如:C:\Users\x\eclipse\jee-oxygen2\eclipse\dropins\jad\jad.exe
点击apply,再点击上面的General-Editors-File Associations
点击*.class,把下面的JadClipse Class File Viewer设置为default默认打开文件
在点击*.class without source,点击下面的Add,把JadClipse Class File Viewer添加进来,再设置为default

完成

现在我们就可以打开项目下所有的包,并阅读它们的源码了~
