  • 2018-02-04
  • v

来来来,往这里看…

配置环境

安装Node.js

下载地址: 地址

安装完成之后打开命令行工具(后文简写为输入命令),输入”node -v”,出现下面的结果,就说明安装成功了!

安装git

下载地址(windows): 地址

输入命令”git version”,出现下面的结果,就说明安装成功了!

安装Hexo

什么是Hexo?

一个快速，简单和功能强大的博客框架

官网如是说

我们先输入命令”npm install hexo-cli -g”初始化Hexo

完成之后输入命令”hexo init blog”,就会在当前目录下新建一个名为”blog”的文件夹(这个目录一般是”C:\用户\你的用户名”)

输入命令”cd blog”进入这个文件夹

输入命令”npm install”安装Hexo

输入命令”hexo server”在本地启动Hexo

完成之后在浏览器上输入http://localhost:4000
就可以看到属于你自己的博客了!!!(按”Ctrl+C”输入”Y”或者关掉命令行工具可以关掉Hexo)

配置GitHub

至今为止,我们的博客只能在我们自己的电脑上浏览
我们只有发布到GitHub上,其他人才能看到我们的博客
我们需要先注册一个Github账号: 地址
注册完之后点击”Start a project”新建一个项目
这里的”Repository name”必须填写为”你注册时的用户名.github.io”,例如”wxwxwxw.github.io”(这将会是我们自己的博客网址)
创建完成之后,打开安装好的Git Bash,输入”git config –global user.name “你注册的用户名””配置用户名

再输入命令”git config –global user.email “你注册的邮箱””配置邮箱

最后输入命令”ssh-keygen -t rsa -C “你注册的邮箱””生成密匙(需要按三次回车)

完成之后我们可以在”C:\用户\你的电脑名.ssh”文件夹里面看到id_rsa.pub文件,用记事本打开,把里面的内容(也就是密匙)全部复制下来,稍后我们会用到

现在我们要回到浏览器上,登录Github,点击头像,打开Settings,我们会看到”SSH and GPG keys”
点击它,再点击右边的”New SSH key”
Title随便写,Key粘贴我们刚刚复制的密匙,然后点击”Add SSH key”
