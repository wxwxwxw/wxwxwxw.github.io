什么是REST(RESTful)

REST（Representational State Transfer）表现层状态转换是Roy Thomas Fielding博士于2000年在他的博士论文中提出来的一种万维网软件架构风格，目的是便于不同软件/程序在网络（例如互联网）中互相传递信息。是根基于超文本传输协议(HTTP)之上而确定的一组约束和属性，是一种设计提供万维网络服务的软件构建风格。

所以RESTful可以翻译成REST风格化。

(String),new String(),toString,String.valueOf

  • March 24, 2018
  • v

(String)Object

强制类型转换,只有当Object引用了String类型时,运行才不会出错
注意:可以使用(String)null,但还是返回null.而不是”null”

new String(param)

生成一个新的String对象,可以传入的param参数:
byte[] bytes
char[] value
String original
StringBuffer buffer
StringBuilder builder

toString

调用Object.toString()方法,返回该对象的字符串表示,通常该字符串由类名+”@”+对象的哈希码组成
但是调用toString()方法的对象不能为null,否则会抛出异常:java.lang.NullPointerException
